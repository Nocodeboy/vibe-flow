## Valores de Diseño

1. **Premium pero accesible**: Estética de alta gama sin ser elitista
2. **Dark mode first**: Diseño oscuro como base
3. **Animaciones con propósito**: Movimiento que mejora UX
4. **Glassmorphism**: Efectos de vidrio sutiles
5. **Responsive perfecto**: Mobile-first approach
