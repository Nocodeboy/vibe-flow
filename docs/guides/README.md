## Resumen de Guías

### Estilos y Tailwind

El proyecto usa Tailwind CSS con configuración personalizada:
- Colores de marca (primary, background, surface)
- Tipografías (Space Mono, Inter)
- Componentes predefinidos

### Animaciones

Framer Motion 12 para todas las animaciones:
- Variants reutilizables
- Stagger effects
- Scroll-triggered animations
- Gestures (hover, tap, drag)

### Accesibilidad

Cumplimiento WCAG 2.1 nivel AA:
- Skip links
- Focus management
- Reduced motion
- ARIA attributes

### Rendimiento

Optimizaciones implementadas:
- Code splitting
- Lazy loading
- Image optimization
- Memoization

### Seguridad

Medidas de seguridad:
- Validación de formularios
- Protección XSS
- Security headers
- Rate limiting
