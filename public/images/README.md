# Estructura de Assets para Vibe Flow

## Carpetas Creadas

```
public/images/
├── hero/           ← Fondo hero, partículas, video loops
├── blog/           ← Imágenes destacadas de artículos
├── personas/       ← Ilustraciones de target audience
├── projects/       ← Mockups y screenshots de case studies
├── team/           ← Fotos del equipo
├── icons/          ← Iconografía custom SVG
├── membership/     ← Ilustraciones de beneficios
└── textures/       ← Noise, gradients, patterns
```

## Formatos Recomendados

| Tipo | Formato | Tamaño |
|------|---------|--------|
| Fotos | WebP | 1200x800 |
| Ilustraciones | SVG o WebP | Variable |
| Icons | SVG | 64x64 |
| Texturas | PNG/WebP | 512x512 tile |
